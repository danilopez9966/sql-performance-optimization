# SQL Performance Optimization

A practical repository focused on **SQL query optimization, indexing strategies, execution plans and performance analysis** using **MySQL and PostgreSQL**.

This project is designed to explore how database queries can be improved through better structure, indexing and analysis tools such as `EXPLAIN` and `EXPLAIN ANALYZE`.

---

## About This Repository

This repository contains examples of slow queries, optimized versions, indexing experiments and performance notes.

The goal is to understand not only how to write SQL queries, but how to make them **faster, cleaner and more efficient**.
